Рабочий класс - gcConfig Предки: gcObject -> gcDynList<gcConfigItem> Методы и конструкторы класса:
- bool LoadFile(gcString Filename) - Загрузить файл конфигурации. Результат - успех/неудача операции.
- bool SaveFile(gcString Filename) - Сохранить файл конфигурации. Результат - успех/неудача операции. Класс конфигурации. Хранит информацию о параметрах файла конфигурации, позволяет запись и загрузку из текстового файла. Заголовок: gcconfig.h
