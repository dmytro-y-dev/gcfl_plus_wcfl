Рабочий класс - gcCmdParser Предки: gcObject Методы и конструкторы класса:
- void ParseCmdLine(gcString &sCmdLine, const uint16 &uParseFrom) - "Пропарсить" коммандную строку sCmdLine начиная с uParseFrom-символа.

- bool IsParamExists(gcString &sParamName) - Проверяет, существует-ли параметр с именем sParamName.

- gcString GetValue(const uint16 &uParamID) - Возвращает значение параметра под индексом uParamID.

- gcString GetParam(const uint16 &uParamID) - Возвращает название параметра под индексом uParamID.

- gcString GetValue(gcString &sParamName) - Возвращает значение параметра c именем sParamName.

- gcString GetCommand() - Возвращает комманду, применяемую к параметрам.
- uint16 GetParamsCount() - Возвращает кол. параметров коммандной строки. Парсер коммандной строки. В реальных условиях (создание меню коммандной строки) можно использовать только лишь воспользовавшись спец. макросами. Заголовок: gccmdparser.h
