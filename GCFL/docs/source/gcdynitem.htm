Вспомогательный класс - gcDynItem Предки: gcObject Методы и конструкторы класса:
- gcDynItem(gcDynItem *itmPrev, const bool &bAutoRemove) - Конструктор с указанием параметров
- gcDynItem * GetNext(void) - Получить указатель на следующий элемент списка
- gcDynItem * GetPrev(void) - Получить указатель на предыдущий элемент списка
- uint32 GetIndex(void) - получить номер индекса элемента в связном списке
- void SwapWithNext(void) - поменять следующий и текущий элементы местами в списке
- void SwapWithPrev(void) - поменять предыдущий и текущий элементы местами в списке Члены класса:
- objClass * Content - Указатель на присвоенный объект
Вспомогательный класc для класса gcDynList. Используется для хранения указателя на объект, а также указателя на следующий и предидущий элементы связного списка. Заголовок: gclist.h
