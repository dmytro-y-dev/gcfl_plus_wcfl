Вспомогательный класс - gcConfigItem Предки: gcObject Методы и конструкторы класса:
- gcConfigItem(gcString sName, gcString sValue) - Конструктор с указанием параметров
- void CleanName() - Удалить все пробелы, табы и т.п. из имени параметра.
- void CleanValue() - Удалить все пробелы, табы и т.п. из значения параметра
- void CleanStr(gcString &str) - Удалить все пробелы, табы и т.п. из определенной строки Члены класса:
- gcString Name - Название параметра
- gcString Value - Значение параметра Вспомогательный класc для класса gcConfig. Используется для хранения имени и значения параметра конфигурации. Один элемент - один параметр - один объект gcConfigItem. Заголовок: gcconfig.h
