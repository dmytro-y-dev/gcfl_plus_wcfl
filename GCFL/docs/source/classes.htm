Классы бывают трех типов: базовые, вспомогательные и рабочие.

Базовый класс - это класс, который является самым первым для какой-либо ветви классов. Класс gcObject является базовым классом для всех остальных классов GCFL

Вспомогательный класс - это класс, который не закладывает основу для других классов. Примером такого класса являются gcConfigItem и gcDynItem. В отличии от базовых классов, эти классы не закладывают основу для развития других классов, т.е. не предусматривают дальнейшее наследование и полиморфизм, и являются самодостаточными классами.

Рабочий класс - это класс, который дает программисту выполнять определенные операции (например, считать/записать файлы определенных форматов).
